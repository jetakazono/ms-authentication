# Microserviço de autenticação com Nodejs
##### Este é um projeto desenvolvido durante algumas lives para dissiminação de conhecimento dentro da DIO.
Composição do nosso projeto Endpoints Base: 

Usuários 
* GET /users 
* GET /users/:uuid
* POST /users 
* PUT /users/:uuid 
* DELETE /users/:uuid 

Autenticação 
* POST /token 
* POST /token/validate
